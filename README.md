# Express Basic Server

This project is a simple Node.js web server built using Express.  
It demonstrates basic routing, conditional routing, dynamic routes, query string handling, and error handling.

## Features
- Basic routes (`/`, `/about`)
- Conditional routing (`/foo`)
- Regular expression routes
- Dynamic URL parameters
- Query string handling
- Custom 404 error handling

## Technologies Used
- Node.js
- Express.js
