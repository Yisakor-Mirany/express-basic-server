# Express Basic Server

This project is a simple Node.js web server built using Express. It demonstrates core Express concepts including basic routing, conditional routing, regular expression routes, dynamic URL parameters, query string handling, and custom error handling.

---

## Features

- Basic routes (`/`, `/about`)
- Conditional routing using `next()` (`/foo`)
- Regular expression routes
- Dynamic URL parameters
- Query string handling
- Custom 404 error handling

---

## Technologies Used

- Node.js
- Express.js
